# tandemclub — Web Portal UI kit

The leader-facing planning tool: **where club leaders plan trips.** Desktop-first,
cleaner and more utilitarian than the app — but still warm, never clinical.

## What's here
- `index.html` — the **Plan a trip** wizard: a left rail of steps (Name it → When
  → Itinerary → Money & rides → Publish) and a main form panel. Click any step;
  the budget splitter, ride-share, and three-way visibility publish screen are
  all interactive.

## Source of truth
Recreated from `tandemclub/apps/web/Prototypes-web/Wireframe 0.2/` (greyscale
desktop wizard), re-skinned into the Golden Hour palette.

## Components used
Button, Input, Segment, Toggle, Stepper, Card, Tag.
